# Crud app
This is a simple crud app that uses nextjs, Typescript, MongoDB to store data. It also uses TailwindCSS for styling.

## Tech stack used
- __NextJs__
- __React__
- __MongoDB__
- __TailwindCSS__
- __Typescript__


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
