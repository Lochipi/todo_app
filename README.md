# Crud app
This is a simple crud app that uses nextjs, Typescript, MongoDB to store data. It also uses TailwindCSS for styling.

## Stack used
- Nextjs
- Typescript
- MongoDB
- TailwindCSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
